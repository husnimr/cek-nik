# Cek NIK KTP

![GitHub repo size](https://img.shields.io/github/repo-size/husnimr/cek-nik)
![GitHub stars](https://img.shields.io/github/stars/husnimr/cek-nik?style=social)
![GitHub license](https://img.shields.io/github/license/husnimr/cek-nik)

Aplikasi web sederhana untuk mengecek informasi dasar dari **Nomor Induk Kependudukan (NIK)** berdasarkan format kode administratif Indonesia.

---

## ⚠️ Disclaimer

Aplikasi ini **hanya digunakan untuk mengecek informasi dasar dari NIK**, seperti:

* Wilayah administratif
* Tanggal lahir
* Jenis kelamin

❗ **Aplikasi ini tidak digunakan untuk mengidentifikasi identitas seseorang secara menyeluruh.**
Data yang ditampilkan merupakan hasil parsing dari struktur NIK, **bukan dari database kependudukan resmi pemerintah**.
